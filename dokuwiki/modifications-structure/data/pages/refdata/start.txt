====== Reference Data ======

Reference data is data that defines a set of permissible values to be used by other data fields. Reference data gains in value when it is widely re-used and widely referenced.

Only concerned persons should update this section. More information on reference data can be found [[technologies:dokuwiki:guidelines:working-with-reference-data:start|here]]. You can always [[mailboxes:ndcc:start|contact us]] if you need any assistance on this subject.
